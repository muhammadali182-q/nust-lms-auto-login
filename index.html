NUST LMS Auto Login

Stop typing LMS credentials again and again. This extension automatically logs you into NUST LMS portal with one click.

⬇ Download Extension

How to Use ?

  1. Click Download Button Above
  2. ZIP file will download automatically
  3. Right click ZIP → Extract All
  4. Open Chrome
  5. Go to → chrome://extensions/
  6. Turn ON Developer Mode (top right)
  7. Click Load Unpacked
  8. Select extracted folder
  9. Add your credentails in puzzle icon on chrome (top right)
  10. Open NUST LMS → Extension will auto login
