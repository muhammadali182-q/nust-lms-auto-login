NUST LMS AUTO LOGIN

Stop wasting time typing credentials. This powerful extension automatically logs you into NUST LMS portal with a single click.

⬇ DOWNLOAD EXTENSION

HOW TO USE

  • Click the Download Extension button
  • ZIP file will download to your computer
  • Right-click the ZIP file and select Extract All
  • Open Google Chrome browser
  • Navigate to chrome://extensions/
  • Enable Developer Mode (toggle in top-right corner)
  • Click Load Unpacked button
  • Select the extracted folder from step 3
  • Click the puzzle icon in top-right and add your LMS credentials
  • Open NUST LMS website - extension will auto-login automatically!
