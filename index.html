NUST LMS AUTO LOGIN

Stop wasting time typing credentials. This powerful extension automatically logs you into the NUST LMS portal with a single click.

⬇ DOWNLOAD EXTENSION

HOW TO USE

  • Click the Download Extension button
  • ZIP file will download to your computer
  • Right-click ZIP file and select Extract All
  • Open Google Chrome browser
  • Navigate to chrome://extensions/
  • Enable Developer Mode (top-right toggle)
  • Click Load Unpacked button
  • Select extracted folder
  • Click puzzle icon and add LMS credentials
  • Open NUST LMS website and auto login will work
