NUST LMS Auto Login

This Chrome extension automatically logs you into the NUST LMS portal. It saves time by securely filling and submitting login credentials.

Download Extension

Setup Instructions (Precisely)

  1. Download the ZIP file from the GitHub link.
  2. Extract the ZIP file using WinRAR or any extractor.
  3. Open Google Chrome.
  4. Go to: chrome://extensions/
  5. Enable Developer Mode (Top Right Corner).
  6. Click on Load Unpacked.
  7. Select the extracted extension folder.
  8. Extension will be installed automatically.
  9. Login once to save credentials if required.
